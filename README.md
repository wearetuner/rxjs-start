## RxJS 프로그래밍 리뷰
Reactive 프로그래밍에 대해 익숙해지자

## Installation

```sh
$ npm install --save rxjs-start
```

## Usage

```js
const rxjsStart = require('rxjs-start');

rxjsStart('Rainbow');
```
